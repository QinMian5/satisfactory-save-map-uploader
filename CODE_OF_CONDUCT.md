# Code of Conduct

This project is maintained as a small, focused open-source project.

## Expected Behavior

- Be respectful and direct.
- Keep discussion focused on the project.
- Assume good intent, but accept maintainer decisions about scope and risk.
- Do not pressure maintainers or contributors to share private files, real save data, credentials, or personal information.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Posting private information without permission.
- Publicly disclosing security issues before the maintainer has had time to review them.
- Repeatedly pushing unrelated scope after a maintainer has declined it.

## Enforcement

The maintainer may edit or delete comments, close issues, block users, or reject contributions that violate this code of conduct.

Report serious conduct concerns through the same private path described in `SECURITY.md` when public discussion would expose sensitive information.
